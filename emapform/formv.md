# emapForm 垂直&水平 表单

### 初始化参数说明

```js
$('#horizonEditForm').emapForm({
  ...
  model: "v",  //表单布局方式; 只在非只读表单中生效
  ...
});
```

#### 提示：
model参数含义
* 'h' = horizontal 水平布局
* 'v' = vertical 垂直布局 
* 't' = table 表格布局

此处设置cols参数是没有效果的
